[Main]
Name=ticable_set_hfc
Type=Function
Header=cabl_int.h
Definition=void ticable_set_hfc(@TicableHfc@ hfc)

[Parameters]
hfc : flow control.

[Summary]
Enable/Disable hardware flow control for the high-speed AVRlink cable.

[Description]
When enabled, the ticables library use RTS/CTS control line for controling the
data flow. This option may improve quality of transfers.

The change will come into effect on the next call to @ticable_set_cable@.

Note: if you need to change more than one link cable parameter at a time, you
should consider the @ticable_set_param@ function.
