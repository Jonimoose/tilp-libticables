[Main]
Name=check
Type=Function
Header=cabl_int.h
Definition=int (*check) (@TicableStatus@* status)

[Parameters]
status : an informational value.
Returns : always 0.

[Summary]
Used for checking the status of a put/get byte.

[Description]
This function is primarily used by emulators. It allows you to check whether
a byte is available on the link cable. This function is non-blocking contrary
to @TicableLinkCable.get@.

When a byte is available for reading, STATUS_RX is returned.
When the byte finished to be sent, STATUS_TX is returned.
Else, STATUS_NONE is returned.
