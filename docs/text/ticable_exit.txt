[Main]

Type=Function
Header=cabl_int.h
Definition=int ticable_exit (void);

[Parameters]
Returns : always 0.

[Summary]
Release the ticables library.

[Description]
This function must be called just before exiting your program. It releases any
allocated resources and close some handles.
